### Mercado Libre Challenge Project 

#### Requisitos

- Nodejs

#### Instalación

Correr los siguientes comandos en orden:

```
npm install
npm run dev:install
```

#### Inicio

```
npm start
```

#### Documentación

Para abrir la documentación del front ejecutar el siguiente comando

```
npm run doc:client
```

#### Testing

Para correr los test del front ejecutar el siguiente comando

```
npm run test:client
```
